# OpenVibely User Guides

These are UI guides for day-to-day use of OpenVibely. They explain what each page is for, why you would use it, and how to complete common workflows.

These are not API docs.

## Start Here

If this is a fresh install:

1. Configure at least one model in `/models`.
2. Create or review agents in `/agents` (optional, but recommended).
3. Create tasks in `/tasks` or use `/chat` to orchestrate work.
4. Tune capacity in `/workers` if tasks queue for too long.
5. Add schedules in `/schedule` for recurring or timed work.
6. Connect channels in `/channels` (Telegram/Slack/GitHub) if needed.
